# Movito Group — Irresistible Offer Architecture
## Hormozi Value Equation Applied

---

## Value Equation

```
Value = (Dream Outcome × Perceived Likelihood) ÷ (Time Delay × Effort)

Dream Outcome:          Pasivní příjem z nemovitostí bez operativních starostí    = 9/10
Perceived Likelihood:   LTV 50-70%, risk assessment, track record, transparentnost = 7/10
Time Delay:             Čtvrtletní výplaty (ne roční), 1-3 roky                    = 2/10 (low = good)
Effort:                 Zero management, jednoduchý 5-krokový proces              = 2/10 (low = good)

Score = (9 × 7) ÷ (2 × 2) = 63/4 = 15.75 → SILNÝ (vše nad 5 je dobré)
```

## Offer Name (M.A.G.I.C. Framework)

**Primární:** "Realitní výnos"
- Magnetic: jasný, profesionální
- Avatar: investoři hledající nemovitostní výnosy
- Goal: pasivní příjem
- Interval: implikovaný (roční/čtvrtletní)
- Container: investiční program

**Alternativa:** "Nemovitostní renta" — evokuje stabilní, pravidelný příjem

## Structural Protective Elements (NE "záruky/garance")

1. **LTV ochranný buffer** — max 70% z risk-assessed hodnoty (ne tržní)
2. **Transparentní reporting** — čtvrtletní report s detaily nemovitostí, LTV, platebním statusem
3. **SPV oddělení** — investice v dedikované společnosti (s právním disclaimerem o limitech)
4. **Due diligence proces** — nezávislý odhad ceny třetí stranou

## Tone & Language Rules

```
✅ POUŽÍT:
- Formální "Vy/Vám" všude
- Specifická čísla ("9,7 % za poslední rok", ne "až 10 %")
- Profesionální ale srozumitelný jazyk
- Transparentnost o rizicích
- "Cílový výnos" (ne "garantovaný" nebo "typický")

❌ NEPOUŽÍVAT:
- Tykání ("ty/tebe")
- Casual/hype jazyk ("bez práce", "bohatí Češi")
- Přehnané sliby
- Fake urgence
- Srovnání s konkrétními konkurenty jménem
- Slovo "záruka/garance" v kontextu výnosů
```
