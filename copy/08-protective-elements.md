# Sekce 8: OCHRANNÉ PRVKY (dříve "Guarantees")

**PRÁVNÍ POZNÁMKA:** Nepoužívat slova "záruka" nebo "garance"
v kontextu výnosů nebo návratnosti investice. Používat
"ochranné prvky" nebo "strukturní ochrana".

---

## Nadpis sekce

```
Jak je Váš kapitál chráněn
```

---

## Ochranný prvek 1: Nemovitostní zajištění

```
┌─────────────────────────────────────────────────────┐
│  OCHRANNÝ PRVEK #1                                   │
│  Zajištění reálnou nemovitostí                       │
│                                                      │
│  Každá investice je zajištěna konkrétní nemovitostí. │
│  Nepůjčujeme víc než 70 % z konzervativně           │
│  stanovené hodnoty nemovitosti.                      │
│                                                      │
│  Příklad:                                            │
│  · Tržní hodnota nemovitosti:   13 000 000 Kč       │
│  · Náš interní (risk) odhad:    10 000 000 Kč       │
│  · Max. výše financování:        7 000 000 Kč       │
│  · Bezpečnostní polštář:         6 000 000 Kč (46%) │
│                                                      │
│  I při výrazném poklesu ceny zůstává Vaše            │
│  investice kryta hodnotou nemovitosti.               │
│                                                      │
│  Ocenění provádí nezávislý odhadce.                  │
└─────────────────────────────────────────────────────┘
```

## Ochranný prvek 2: Oddělení majetku (SPV)

```
┌─────────────────────────────────────────────────────┐
│  OCHRANNÝ PRVEK #2                                   │
│  Oddělení majetku investorů                          │
│                                                      │
│  Vaše investované prostředky jsou uloženy            │
│  v samostatné společnosti (SPV — Special Purpose     │
│  Vehicle), oddělené od provozních prostředků         │
│  Movito Group.                                       │
│                                                      │
│  To znamená:                                         │
│  · Vaše peníze nejsou na běžném účtu Movito          │
│  · Jsou účelově vázány na konkrétní projekt          │
│  · V případě potíží Movito Group zůstávají           │
│    prostředky v SPV chráněny                         │
│                                                      │
│  ⚠️ SPV struktura snižuje, ale zcela neeliminuje    │
│  riziko insolvence. Podrobnosti ve smluvní           │
│  dokumentaci.                                        │
└─────────────────────────────────────────────────────┘
```

## Ochranný prvek 3: Transparentní reporting

```
┌─────────────────────────────────────────────────────┐
│  OCHRANNÝ PRVEK #3                                   │
│  Plná transparentnost                                │
│                                                      │
│  Každé čtvrtletí obdržíte detailní report:          │
│                                                      │
│  · Aktuální stav nemovitosti (fotodokumentace)       │
│  · Aktuální LTV (poměr půjčky k hodnotě)           │
│  · Platební morálka dlužníka                         │
│  · Výkon Vašeho portfolia vs. benchmark              │
│  · Plánované kroky na další čtvrtletí                │
│                                                      │
│  Víte přesně, kde jsou Vaše peníze a jak             │
│  se jim daří. Žádné černé skříňky.                  │
└─────────────────────────────────────────────────────┘
```

## Ochranný prvek 4: Due diligence proces

```
┌─────────────────────────────────────────────────────┐
│  OCHRANNÝ PRVEK #4                                   │
│  Důkladný výběr projektů                             │
│                                                      │
│  Než Vám projekt nabídneme, projde:                 │
│                                                      │
│  1. Právní prověrka nemovitosti (věcná břemena,      │
│     zástavní práva, vlastnická historie)             │
│  2. Nezávislé ocenění třetí stranou                  │
│  3. Posouzení bonity dlužníka                        │
│  4. Analýza lokality a tržního potenciálu            │
│  5. Interní risk assessment (konzervativní ocenění)  │
│                                                      │
│  Schvalujeme pouze projekty, kde je riziko           │
│  v akceptovatelné míře a zajištění dostatečné.       │
│                                                      │
│  [Doplnit: „Z X posuzovaných projektů schválíme      │
│  pouze Y %" — pokud existují data]                   │
└─────────────────────────────────────────────────────┘
```

---

## Shrnutí ochranných prvků

```
Vaše investice je chráněna na 4 úrovních:

  ✓ Reálná nemovitost jako kolaterál (LTV max 70 %)
  ✓ Oddělená právní struktura (SPV)
  ✓ Čtvrtletní transparentní reporting
  ✓ Důkladný výběrový proces každého projektu

Žádná investice není bez rizika. Ale děláme maximum
pro to, aby Vaše riziko bylo měřitelné a řízené.
```
